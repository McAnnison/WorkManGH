# Assets Directory

This directory contains app assets like icons and splash screens.

For a production app, you would need to add:
- icon.png (1024x1024)
- adaptive-icon.png (1024x1024)
- splash.png (2048x2048)
- favicon.png (48x48)

You can generate these assets using:
- Figma or similar design tools
- Online icon generators
- Expo's asset generation tools

For now, the app will use default Expo icons.
